This project implements Generative Adversarial Networks (GANs) which consist of two neural networks competing against each other: a generator that creates fake data and a discriminator that tries to distinguish real from fake data. It includes simple GANs, Wasserstein GANs with clipping and gradient penalty, and face generation capabilities.
