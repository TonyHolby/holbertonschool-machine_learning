# <p align="center">Temporal Difference</p>

![Temporal Difference image](temporal_difference.png)

##  Description

This project implements Temporal Difference (TD) learning methods including Monte Carlo methods, TD(位) algorithm and SARSA(位). These algorithms enable an agent to learn optimal policies through value estimation and policy improvement.

## Technologies

python3 (version 3.9)  
numpy (version 1.25.2)

```
git clone https://github.com/TonyHolby/holbertonschool-machine_learning.git
cd reinforcement_learning/temporal_difference/
```
